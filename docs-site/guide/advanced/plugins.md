# 插件系统

Polaris 支持插件系统，可以扩展 AI 的能力范围。通过插件市场浏览、安装和管理插件。

在 **设置 → 插件** 中管理。

## 界面布局

插件管理页面采用三栏布局：

| 栏目 | 说明 |
|------|------|
| 左侧：分类导航 | 按类别浏览插件 |
| 中间：插件列表 | 显示可用/已安装的插件，支持虚拟滚动 |
| 右侧：插件详情 | 选中插件的详细信息、配置和操作按钮 |

## 安装插件

1. 在左侧分类中选择要浏览的类别
2. 在中间列表中找到目标插件
3. 点击插件查看详情
4. 点击 **安装** 按钮
5. 安装完成后插件自动可用

## 管理已安装插件

| 操作 | 说明 |
|------|------|
| 启用/禁用 | 切换插件的活跃状态，禁用后插件功能不可用 |
| 卸载 | 完全移除插件 |
| 更新 | 检查并安装插件的新版本 |
| 配置 | 部分插件支持自定义配置（如 MCP 服务器地址） |

## 插件市场

### 添加市场

支持添加多个插件市场源：

1. 在市场管理区域点击「添加市场」
2. 输入市场地址
3. 保存后即可浏览该市场的插件

### 市场管理

- 查看已添加的市场列表
- 移除不需要的市场
- 刷新市场内容

## 插件作用域

| 作用域 | 说明 |
|--------|------|
| 用户级 | 所有工作区共享，全局生效 |
| 项目级 | 仅在当前工作区生效 |
| 本地 | 仅在本地环境生效 |

## MCP 服务器配置

部分插件会注册 MCP 服务器，扩展 AI 的工具能力：

- 安装插件后，相关 MCP 服务器自动注册
- 在插件详情中可以查看和修改 MCP 服务器配置
- AI 对话时会自动使用已启用的 MCP 工具

# 快捷键参考

以下是 Polaris 中所有可用的快捷键。

## 全局快捷键

| 快捷键 | 操作 |
|--------|------|
| <span class="shortcut">F12</span> | 打开/关闭开发者工具 (DevTools) |
| <span class="shortcut">Shift</span> + <span class="shortcut">Ctrl</span> + <span class="shortcut">R</span> | 打开文件搜索面板 |
| <span class="shortcut">Escape</span> | 关闭弹窗、下拉菜单、建议列表 |

## AI 对话

| 快捷键 | 操作 |
|--------|------|
| <span class="shortcut">Enter</span> | 发送消息 |
| <span class="shortcut">Shift</span> + <span class="shortcut">Enter</span> | 换行 |
| <span class="shortcut">Ctrl</span> / <span class="shortcut">Cmd</span> + <span class="shortcut">F</span> | 搜索对话内容 |
| <span class="shortcut">Tab</span> | 接受自动补全建议 |
| <span class="shortcut">↑</span> / <span class="shortcut">↓</span> | 在自动补全建议中导航 |

## 文件浏览器

| 快捷键 | 操作 |
|--------|------|
| <span class="shortcut">↑</span> / <span class="shortcut">↓</span> | 在文件树中导航 |
| <span class="shortcut">Enter</span> | 展开/折叠文件夹，或打开文件 |
| <span class="shortcut">Shift</span> + <span class="shortcut">Ctrl</span> + <span class="shortcut">R</span> | 全局文件搜索 |

## 代码编辑器

基于 CodeMirror 6，支持标准编辑器快捷键：

| 快捷键 | 操作 |
|--------|------|
| <span class="shortcut">Ctrl</span> / <span class="shortcut">Cmd</span> + <span class="shortcut">F</span> | 文件内搜索 |
| <span class="shortcut">Ctrl</span> / <span class="shortcut">Cmd</span> + <span class="shortcut">H</span> | 搜索并替换 |
| <span class="shortcut">Ctrl</span> / <span class="shortcut">Cmd</span> + <span class="shortcut">S</span> | 保存文件 |
| <span class="shortcut">Ctrl</span> / <span class="shortcut">Cmd</span> + <span class="shortcut">Z</span> | 撤销 |
| <span class="shortcut">Ctrl</span> / <span class="shortcut">Cmd</span> + <span class="shortcut">Shift</span> + <span class="shortcut">Z</span> | 重做 |

## 对话内容交互

| 快捷键 | 操作 |
|--------|------|
| <span class="shortcut">Enter</span> / <span class="shortcut">Space</span> | 展开/折叠工具调用、思考块 |
| <span class="shortcut">Enter</span> | 确认对话框 |
| <span class="shortcut">Escape</span> | 取消对话框 |

## 标签页

| 快捷键 | 操作 |
|--------|------|
| <span class="shortcut">Ctrl</span> / <span class="shortcut">Cmd</span> + <span class="shortcut">W</span> | 关闭当前标签页 |

## 语音命令

开启语音输入后，可使用语音命令：

| 命令 | 操作 |
|------|------|
| 「发送消息」 | 发送当前输入框内容 |
| 「清空输入框」 | 清空对话输入框 |
| 「中断对话」 | 中断 AI 正在生成的回复 |
| 「撤回最后输入」 | 撤回最后一条用户消息 |

## 提示

- macOS 用户使用 <span class="shortcut">Cmd</span> 替代 <span class="shortcut">Ctrl</span>
- 快捷键在输入框获得焦点时优先响应输入框操作
- 部分快捷键可通过系统设置自定义

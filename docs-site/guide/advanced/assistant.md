# 独立助手

Polaris 内置独立助手面板，使用独立的 LLM 后端，不依赖 Claude Code CLI。适合快速问答和日常辅助。

通过 ActivityBar 的 **助手图标** 打开助手面板。

## 与 AI 对话的区别

| 维度 | AI 对话面板 | 独立助手面板 |
|------|-----------|------------|
| 后端 | Claude Code CLI | 独立 LLM API |
| 工具调用 | 支持文件操作、命令执行等 | 仅对话，无工具调用 |
| 适用场景 | 代码开发、文件操作 | 快速问答、知识查询 |
| 配置 | 使用 CLI 配置 | 独立配置 API |

## 启用助手

1. 打开 **设置 → 助手**
2. 开启「启用助手」开关
3. 配置 LLM 后端参数

## LLM 配置

| 参数 | 说明 |
|------|------|
| Base URL | API 服务地址（兼容 OpenAI 格式） |
| API Key | 认证密钥 |
| 模型 | 选择具体的模型名称 |
| Max Tokens | 单次回复最大 Token 数 |
| Temperature | 回复随机性（0~1，越高越随机） |

支持任何兼容 OpenAI API 格式的服务商（如 DeepSeek、Groq、自建服务等）。

## 系统提示词

助手支持独立的系统提示词配置：

| 模式 | 说明 |
|------|------|
| 追加 | 在默认提示词后追加自定义内容 |
| 替换 | 完全替换为自定义内容 |

可以根据不同用途设置不同的系统提示词，例如：

```
你是一个前端开发专家，擅长 React 和 TypeScript。
请用简洁清晰的方式回答问题。
```

## Claude Code 会话管理

助手面板还可以管理当前运行的 Claude Code 会话：

- 查看正在运行的 Claude Code 会话列表
- 监控会话状态
- 快速切换到对应会话

## 使用场景

| 场景 | 说明 |
|------|------|
| 快速提问 | 不需要文件操作的简单问题，如「JavaScript 的 Promise 怎么用」 |
| 知识查询 | 查询技术文档、API 用法等 |
| 多模型对比 | 使用不同的 LLM 后端，与 AI 对话面板的 Claude 进行对比 |
| 轻量对话 | 不需要启动 Claude CLI 的轻量场景 |

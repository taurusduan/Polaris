# 快捷片段

快捷片段允许用户创建可复用的提示词模板，在对话中通过 `/` 快速调用。

在 **设置 → 快捷片段** 中管理。

## 创建片段

| 字段 | 说明 |
|------|------|
| 片段名称 | 用于 `/name` 快捷调用，不可重复 |
| 描述 | 片段的简要说明 |
| 模板内容 | 使用 <code v-pre>`{{变量名}}`</code> 定义变量占位符 |
| 启用 | 控制片段是否可用 |

## 变量系统

### 自定义变量

在模板内容中使用 <code v-pre>`{{变量名}}`</code> 定义变量。调用片段时会弹出表单填写变量值：

| 变量类型 | 说明 |
|----------|------|
| 单行文本 | 短文本输入 |
| 多行文本 | 长文本输入 |

每个变量可配置：键名、标签、是否必填、默认值、占位提示。

### 自动变量

无需定义即可使用的系统变量：

| 变量 | 说明 |
|------|------|
| <code v-pre>`{{date}}`</code> | 当前日期 |
| <code v-pre>`{{time}}`</code> | 当前时间 |
| <code v-pre>`{{workspaceName}}`</code> | 当前工作区名称 |
| <code v-pre>`{{workspacePath}}`</code> | 当前工作区路径 |

## 使用方式

在 AI 对话输入框中输入 `/` 会弹出快捷片段列表，选择后：

1. 弹出变量填写表单（如有自定义变量）
2. 填写变量值
3. 确认后自动插入到输入框

## 示例

创建一个代码审查片段：

- **名称**：`review`
- **模板内容**：<code v-pre>`请审查以下 {{language}} 代码，关注 {{focus_points}} 方面：\n\n{{code}}`</code>
- **变量**：
  - `language` — 单行文本，默认值 "TypeScript"
  - `focus_points` — 单行文本，默认值 "性能、安全性"
  - `code` — 多行文本，必填

在对话中输入 `/review` 即可快速调用。

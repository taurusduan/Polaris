# Git 版本控制

Polaris 内置 Git 面板，提供常用的版本控制操作。

## 打开 Git 面板

通过 ActivityBar 的 **Git** 图标打开 Git 面板。

## 功能概览

### 变更查看

- 查看 **已修改**、**新增**、**删除** 的文件列表
- 点击文件查看 diff（差异对比视图）
- 支持逐行查看增删内容

### 提交

1. 在文件列表中选择要提交的文件
2. 在底部输入框填写提交信息
3. 点击 **提交** 按钮

### 分支管理

- 查看当前分支
- 切换已有分支
- 创建新分支

::: tip 提交信息规范
建议使用 Conventional Commits 格式：`feat:`, `fix:`, `docs:`, `refactor:` 等前缀。
:::

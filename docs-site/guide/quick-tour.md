# 五分钟快速体验

第一次使用 Polaris？跟着下面的步骤，五分钟内体验核心功能。

## 第一步：确认连接

启动应用后，右侧面板是 AI 对话区。如果你看到欢迎页和输入框，说明 Claude CLI 已连接成功。

如果看到连接页面，请参考 [安装与启动](./getting-started) 完成连接配置。

## 第二步：发送第一条消息

在右侧面板底部的输入框中：

1. 输入 `你好，请介绍一下你自己`
2. 按 <span class="shortcut">Enter</span> 发送
3. AI 会以流式方式实时返回回复

<div class="info-card tip">
  <div class="card-title">提示</div>
  <p>按 <span class="shortcut">Shift</span> + <span class="shortcut">Enter</span> 可以换行输入多行内容，不会触发发送。</p>
</div>

## 第三步：体验文件管理

1. 点击左侧 ActivityBar 的 **Files** 📄 图标
2. 展开文件树，浏览当前工作区的项目文件
3. 双击任意文件，在中间区域打开代码编辑器
4. 编辑器支持 20+ 种语言的语法高亮

## 第四步：让 AI 操作文件

回到 AI 对话，试试发送：

```
请读取 src/ 目录下的文件列表，并简要描述项目结构
```

AI 会自动调用文件读取工具，对话中会显示工具调用卡片，展示读取的文件内容。

## 第五步：查看工具调用详情

AI 在执行操作时会显示工具调用卡片：

1. 找到对话中的工具调用卡片（带有文件图标或命令图标）
2. 点击 **展开** 查看详细信息
3. 可以看到文件差异、命令输出等内容

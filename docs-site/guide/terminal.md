# 终端

Polaris 内置终端面板，基于 xterm.js，提供完整的命令行操作体验。通过 Tauri PTY 后端实现真实的 Shell 交互。

通过 ActivityBar 的 **Terminal** 图标打开终端面板。

## 创建终端会话

面板右上角提供 **+** 按钮创建新的终端会话：

- 支持同时打开多个终端实例
- 每个终端以标签页形式展示，可自由切换和关闭
- 新建终端默认使用系统 Shell

| 操作系统 | 默认 Shell |
|----------|-----------|
| Windows | PowerShell / CMD |
| macOS | zsh / bash |
| Linux | bash / zsh |

## 终端操作

### 基本交互

| 操作 | 说明 |
|------|------|
| 命令执行 | 输入命令后按 <span class="shortcut">Enter</span> 执行 |
| 多行输出 | 支持长输出滚动查看 |
| 自适应尺寸 | 面板宽度变化时自动调整终端列数和行数 |
| 拖拽调整 | 通过面板边界拖拽调整终端面板的宽度 |

### 会话管理

| 操作 | 说明 |
|------|------|
| 新建终端 | 点击右上角 **+** 按钮 |
| 切换终端 | 点击终端标签页切换到对应会话 |
| 关闭终端 | 点击标签页上的关闭按钮 |

<div class="info-card tip">
  <div class="card-title">提示</div>
  <p>关闭终端会话不会中断正在运行的进程（后台进程会继续执行），但你会无法再看到该终端的输出。</p>
</div>

## 滚动与历史

| 功能 | 说明 |
|------|------|
| 滚动缓冲区 | 默认保留 10,000 行历史输出 |
| 向上滚动 | 查看已滚出屏幕的命令输出 |
| 搜索历史 | 在终端中使用 <span class="shortcut">↑</span> / <span class="shortcut">↓</span> 箭头浏览历史命令 |

## 链接识别

终端自动识别输出中的 URL 链接，鼠标悬停时高亮显示，点击可直接在浏览器中打开。

## 使用场景

### 与 AI 对话配合

终端和 AI 对话面板可以同时使用：

1. 在 AI 对话中让 AI 分析代码或生成命令
2. 在终端中手动执行和验证命令
3. 将终端输出截图或复制粘贴回 AI 对话，让 AI 进一步分析

### 常见操作示例

| 场景 | 命令示例 |
|------|----------|
| 查看项目状态 | `git status` |
| 安装依赖 | `pnpm install` |
| 运行测试 | `pnpm test` |
| 查看日志 | `tail -f logs/app.log` |
| 检查端口占用 | `netstat -ano \| findstr :3000` (Windows) |

## 注意事项

- 终端面板宽度影响每行可显示的字符数，过窄可能导致长命令折行，建议保持足够宽度
- 终端使用 VS Code Dark 配色主题，与 Polaris 整体风格一致
- 如果终端无响应，尝试关闭并重新创建一个终端会话

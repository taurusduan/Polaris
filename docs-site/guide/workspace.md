# 工作区

工作区是 Polaris 管理项目和配置的基本单元。

## 创建工作区

1. 启动时或通过顶栏的工作区选择器创建新工作区
2. 选择一个本地目录作为工作区根路径
3. 工作区会记住该路径下的文件结构和配置

## 切换工作区

通过顶栏的工作区选择器：
- 点击当前工作区名称打开选择器
- 选择已有工作区快速切换
- 或创建新的工作区

## 工作区配置

每个工作区独立维护：

| 配置项 | 说明 |
|--------|------|
| 文件浏览器状态 | 展开的目录、选中的文件 |
| AI 会话列表 | 该工作区下的对话历史 |
| System Prompt | 该工作区的自定义系统提示词 |
| 面板布局 | 面板宽度和显示状态 |

## 数据存储

工作区数据存储在本地：
- 会话历史持久化到本地存储
- 支持跨会话恢复上下文
- 空闲会话自动释放内存，切换时恢复

# 文件管理

Polaris 内置文件浏览器和代码编辑器，无需离开应用即可管理项目文件。

## 文件浏览器

通过 ActivityBar 的 **文件** 图标打开左侧文件浏览器。

### 基本操作

| 操作 | 方法 |
|------|------|
| 打开文件 | 单击文件 |
| 新建文件 | 点击工具栏新建按钮或右键菜单 |
| 新建文件夹 | 点击工具栏新建按钮或右键菜单 |
| 重命名 | 右键 > 重命名 |
| 删除 | 右键 > 删除 |
| 复制路径 | 右键 > 复制路径 |

### 文件搜索

使用全局搜索功能（快捷键）快速定位文件：
- 支持按文件名模糊搜索
- 显示匹配路径和文件类型

## 代码编辑器

双击文件在 CenterStage 区域打开编辑器。

### 功能特性

- **多标签页** — 同时打开多个文件，标签页切换
- **语法高亮** — 支持 20+ 编程语言
- **代码折叠** — 折叠/展开代码块
- **搜索替换** — 文件内查找和替换
- **自动补全** — 上下文感知的代码补全
- **差异对比** — Git 变更的 diff 视图

### 支持的语言

JavaScript, TypeScript, Python, Rust, Go, Java, C++, C, HTML, CSS, JSON, XML, SQL, Markdown, YAML 等。

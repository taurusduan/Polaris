## 快速摘要

### 高优先级功能

1. **MCP 服务器管理** - 独立管理面板，状态监控，配置管理
2. **代理管理** - 代理列表展示，快速切换
3. **认证管理** - 登录状态，多种认证方式
4. **自动模式配置** - 规则编辑，环境配置

### 技术方案

- 复用现有 `SettingsModal` 架构
- 通过 Tauri Command 执行 CLI 命令
- Zustand 状态管理
- 三栏布局风格统一

### 预估工时

- 阶段一（核心功能）：约 11 小时
- 阶段二（增强功能）：约 7.5 小时
- 阶段三（优化完善）：约 5 小时

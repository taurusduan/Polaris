# 03 可视化方案设计 (Visualization Design)

## 1. 项目背景分析

### 1.1 现有架构

Polaris 项目已经具备完善的设置系统和插件管理功能：

**已有组件：**
- `SettingsModal` - 设置模态框，支持 Tab 切换
- `SettingsSidebar` - 设置导航栏
- `PluginTab` - 插件管理 Tab（完整的三栏布局）
- `SessionHistoryPanel` - 会话历史面板

**现有功能：**
- 插件列表、安装、启用/禁用、卸载
- 市场管理（添加、移除、更新）
- 会话历史查看和恢复
- MCP 服务器状态展示

### 1.2 Claude CLI 功能映射

| Claude CLI 功能 | 现有实现 | 需要新增/增强 |
|-----------------|----------|---------------|
| `claude agents` | 无 | 需新增代理管理面板 |
| `claude auth` | 无 | 需新增认证管理面板 |
| `claude mcp` | 部分（插件内展示） | 需增强独立 MCP 管理面板 |
| `claude plugin` | 已有 PluginTab | 可优化增强 |
| `claude auto-mode` | 无 | 需新增自动模式配置 |
| `claude --resume` | 已有 SessionHistoryPanel | 可优化体验 |
| `claude --model` | 无 | 需新增模型选择器 |
| `claude --permission-mode` | 无 | 需新增权限配置 |
| `claude --worktree` | 无 | 需新增 Worktree 管理 |

---

## 2. 设计原则

### 2.1 一致性

- 复用现有 `SettingsModal` 架构
- 保持与 `PluginTab` 相同的三栏布局风格
- 统一使用项目设计系统（颜色、间距、组件）

### 2.2 可发现性

- 关键功能入口明显
- 状态信息清晰可见
- 操作反馈及时

### 2.3 效率优先

- 常用功能一步可达
- 批量操作支持
- 键盘快捷键支持

## 6. 技术方案

### 6.1 CLI 调用方式

通过 Tauri 的 Command 执行 Claude CLI 命令：

```typescript
// src/services/claudeCli.ts
import { Command } from '@tauri-apps/api/shell';

export async function executeClaudeCommand(args: string[]): Promise<string> {
  const command = new Command('claude', args);
  const output = await command.execute();
  return output.stdout;
}

// 获取代理列表
export async function getAgents(): Promise<Agent[]> {
  const output = await executeClaudeCommand(['agents', '--json']);
  return JSON.parse(output);
}

// 获取 MCP 服务器列表
export async function getMcpServers(): Promise<McpServer[]> {
  const output = await executeClaudeCommand(['mcp', 'list']);
  // 解析输出
}
```

### 6.2 状态管理

使用 Zustand 管理各模块状态：

```typescript
// src/stores/agentStore.ts
interface AgentState {
  agents: Agent[];
  loading: boolean;
  error: string | null;
  fetchAgents: () => Promise<void>;
  selectAgent: (id: string) => void;
}
```

### 6.3 错误处理

统一错误处理机制：

```typescript
try {
  await someOperation();
} catch (error) {
  if (error instanceof ClaudeCliError) {
    // 处理 CLI 错误
    toastError(t('errors.cliError'), error.message);
  } else {
    // 处理其他错误
    toastError(t('errors.unknown'), String(error));
  }
}
```

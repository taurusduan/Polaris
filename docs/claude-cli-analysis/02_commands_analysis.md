## 2. auth 子命令

### 2.1 命令结构

```
claude auth [command]

Commands:
  login [options]   登录 Anthropic 账户
  logout            登出
  status [options]  显示认证状态
```

### 2.2 功能测试

**auth status 输出示例：**
```json
{
  "loggedIn": true,
  "authMethod": "oauth_token",
  "apiProvider": "firstParty"
}
```

**auth login 选项：**
- `--claudeai` - 使用 Claude 订阅 (默认)
- `--console` - 使用 Anthropic Console
- `--email <email>` - 预填邮箱
- `--sso` - 强制 SSO 登录

### 2.3 可视化设计

```
┌─────────────────────────────────────────────────────┐
│ 🔐 认证管理                                          │

│ 方式: OAuth Token                                   │
│ 提供商: Anthropic (First Party)                     │
│                                                      │
│ ┌───────────────────────────────────────────────┐  │
│ │ 登录方式                                       │  │
│ │ ○ Claude 订阅 (推荐)                          │  │
│ │ ○ Anthropic Console (API 计费)                │  │
│ │ ○ SSO 企业登录                                 │  │
│ └───────────────────────────────────────────────┘  │
│                                                      │
│ [📧 邮箱登录] [🔄 刷新令牌] [🚪 登出]               │
└─────────────────────────────────────────────────────┘
```

### 2.4 可视化价值

## 3. mcp 子命令

### 3.1 命令结构

```
claude mcp [command]

Commands:
  add [options] <name> <commandOrUrl> [args...]  添加 MCP 服务器
  add-from-claude-desktop [options]              从 Claude Desktop 导入
  add-json [options] <name> <json>               用 JSON 添加服务器
  get <name>                                     获取服务器详情
  list                                           列出所有服务器
  remove [options] <name>                        移除服务器
  reset-project-choices                          重置项目选择
  serve [options]                                启动 MCP 服务器
```

## 5. auto-mode 子命令

### 5.1 命令结构

```
claude auto-mode [command]

Commands:
  config              打印有效配置
  critique [options]  AI 反馈自定义规则
```

### 5.2 功能测试

**auto-mode config 输出：** 返回完整的 allow/soft_deny 规则和 environment 配置

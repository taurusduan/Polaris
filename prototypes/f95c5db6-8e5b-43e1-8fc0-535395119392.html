● 分阶段实施路线
Phase 1 · P0

OpenAI 兼容引擎 + 模型管理面板

实现 OpenAICompatibleEngine（Rust HTTP 客户端 + TS 适配），提供商 CRUD 面板，连接测试，预置模板。扩展 EngineId 枚举，新增 HTTP 会话管理。

Phase 2 · P1

Ollama 本地模型 + Tool Calling 适配

OllamaEngine（REST 直连），模型发现/下载管理，硬件监控。Unified Tool Adapter 抽象层，Prompt 降级模式。MCP 工具定义格式转换器。

Phase 3 · P2

智能路由 + 用量统计

SmartRouter 基于场景/成本/可用性自动选择模型。Token 计量中间件，成本估算引擎，趋势仪表盘。Fallback 策略配置 UI。

✓ 已有架构基础（可直接复用）
